Simple Arcade
Simple Arcade is a project I made in my IT-202: Internet Applications Class. Users play a simple HTML game for points and can register accounts to save their scores. Users can create public and private profiles as well as competitions to challenge other users. There is also a weekly, monthly, and lifetime scoreboard for the game. For users with admin privileges, they can create roles, edit competitions, and assign other users as admins.

Skills Learned: HTML, PHP, CSS, JavaScript, SQL, Bootstrap
